Technical Overview

EDISPHERE is a complete solution for robustly automating complex supply chains, comprising of, several trading partners, ERP systems, and cloud systems, over Internet. It is a modern Electronic Data Interchange (EDI) translation & Web Services integration system that seamlessly integrates your internal business applications with your external partners; mapping business messages in a variety of data formats, such as, X12, EDIFACT, XML, JSON, SAP IDOCs, Proprietary data formats (fixed length, variable length, CSV), etc., and securely exchanging the messages over internet using FTP, WEB SERVICES, etc. It also maps directly to databases.

Unlike most data Integration products, EDISPHERE has built-in productivity features along with rich mapping features, which helps in creating robust and maintainable data integration implementations very quickly. Besides, EDISPHERE is a high-performing scalable engine, enabled with active redundancy features; making it highly suited for 24×7 business intensive data exchange environments.

EDISPHERE data integration platform is available as private instance and as shared instance

EDISPHERE is horizontally oriented (industry-neutral), designed to accommodate the needs of vertical industries such as Automotive, Banking and Financial, Government, Healthcare, Insurance, Manufacturing, Retail, Transportation/Logistics, etc.

Our product development strategy consists of whole-product approach, which places strong emphasis on mapping non-standard and proprietary situations, which are mainstay of modern data integration implementation projects. The result has been the development of smart and innovative features in the product, which are certain to meet the peculiarities of your data integration requirements in a timely, reliable, and cost-effective manner.

Simply stated, EDISPHERE provides the best value in meeting your Data Integration requirement as well as your return on investment (ROI).

Private Instance Model

The Edisphere private instance can be deployed on premise or on cloud as a dedicated virtual machine.

System Requirement:

  1. 4 core CPU, 16GB RAM, 200GB HDD
  2. 64-bit Windows 2019 server
  3. 64-bit MS SQL 2019 Server or Azure SQL (if on Azure cloud)

Pricing: Customized to suit your return on investment (ROI), several different pricing models are offered, including, license fees model, rental model, pay-per-use model, other custom model, like number of trading partners, number of maps, number of messages, etc. Pricing is not based on number of CPU cores or users.

Shared Instance Model

The Edisphere shared instance is hosted on Azure cloud at URL integrateIndia.in, primarily for integrating businesses in India. The shared instance is fully managed by Edisphere.

System Requirement:

  1. Capability in your ERP to import / export files for exchanging with your trading partners.
  2. Internet connection and one or more of several communication options, like, ftp to upload / download messages, web services, etc.

Pricing: Customized to suit your return on investment (ROI), several different pricing models are offered, including, flat monthly charges, pay-per-use model, other custom model, like number of trading partners, number of maps, number of messages, etc. Presently, only flat monthly charges are offered, based on your ROI vis-à-vis your data volume.
