EDISPHERE – Enterprise Data Integration Suite (Executive Overview)

EDISPHERE is a comprehensive enterprise data integration system, seamlessly integrating your internal business applications with your external partners, mapping business messages in a variety of data formats, such as, X12, EDIFACT, XML, JSON, Proprietary data formats (fixed length, variable length, CSV, etc.) and also supports mapping to databases.

Unlike most EDI products, EDISPHERE has built-in productivity features along with mapping features, which helps in creating robust and maintainable EDI implementations very quickly. Besides, EDISPHERE Translator (it’s map execution runtime engine), is a high-performing scalable engine, enabled with active redundancy features, making it highly suited for 24×7 business critical EDI environments.

EDISPHERE is horizontally oriented (industry-neutral), designed to accommodate the needs of vertical industries such as Automotive, Banking and Financial, Government, Healthcare, Insurance, Manufacturing, Retail, Transportation/Logistics, etc.

Our product development strategy consists of whole-product approach, which places strong emphasis on mapping non-standard and proprietary situations which are mainstay of EDI implementation projects. The result has been the development of smart and innovative features in the product, which are certain to meet the peculiarities of your EDI requirements in a timely, reliable and cost-effective manner.

Simply stated, EDISPHERE provides the best value for your EDI Integration requirements.

Seamless EDI Integration

EDISPHERE data integration platform seamlessly integrates with your business application by adapting to the file format or database required by your business application. Your application may be on a different operating system. EDISPHERE communicates with your application using a common database and a common directory/folder, using different communication protocols like FTP, Email, HTTP, MQ Series, etc. as appropriate. It also supports API integration using SOAP / REST based web services.

No programming is necessary for integrating EDISPHERE with your business application if integrated “loosely” in this manner.

EDISPHERE also exposes Application Programming Interfaces (APIs) using Web Services for “tighter” integration with your business application (See white paper – Application Integration using EDISPHERE).

EDISPHERE - SAP Integration using SAP IDOC

EDISPHERE can seamlessly integrate with SAP system using SAP IDOCs (see white paper – SAP integration using EDISPHERE)

EDISPHERE – ERP Integration

All major ERP systems exchange business messages using some hierarchal file format in XML, JSON, CSV, or flat-files. These hierarchal file formats can be easily created using Edisphere Implementor tool and mapped to the desired file format.

EDISPHERE – Cloud Data Integration

All major SaaS providers like Salesforce.com, Netsuite.com, Amazon, etc. publish SOAP / REST based web services for API integration using XML or JSON file formats. EDISPHERE has the capability to create these file formats, import their XML schemas, map them to the desired format and integrate using web services. See white paper – Web Services API Integration using Edisphere.
